Ubuntu es un sistema operativo de software libre y código abierto basado en Debian, diseñado para ser fácil de usar y mejorar la experiencia del usuario.

Sus patrocinadores

Canonical Ltd. es la empresa detrás de Ubuntu, que ofrece el sistema de forma gratuita y recibe contribuciones de la comunidad para mejorar continuamente.

Cada seis meses se publica una nueva versión de Ubuntu, con soporte ofrecido por Canonical durante 9 meses. Las versiones LTS (Long Term Support) reciben soporte durante 5 años en sistemas de escritorio y servidores.

Software incluido

Ubuntu incluye una amplia gama de aplicaciones para tareas cotidianas, entretenimiento, desarrollo y configuración del sistema. Utiliza la interfaz Unity y aplicaciones de GNOME.

Existen versiones extraoficiales mantenidas por la comunidad con diferentes escritorios que pueden instalarse independientemente.

Seguridad y accesibilidad

Ubuntu cuenta con funciones avanzadas de seguridad y políticas como la no activación de procesos latentes por defecto. No incluye un cortafuegos predeterminado, ya que no hay servicios que amenacen la seguridad del sistema.

Para tareas administrativas, se utiliza la herramienta sudo en la línea de comandos para evitar el uso del usuario administrador. Además, Ubuntu se centra en la accesibilidad e internacionalización, utilizando UTF-8 como codificación de caracteres predeterminada.
